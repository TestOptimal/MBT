Build State model that describes AUT behaviors. Use State (node) for AUT state and transition (edge) for actions that change AUT state. Each model must have one Initial State and at least one Final State and all states must be reachable from Initial State.

TestOptimal generates test sequences (test paths) from the model using the selected sequencer to achieve desired test coverage.

Editor has three modes:

  • Edit: view/edit/position state and transition properties, and full model execution.
  • O ➜ Add: add states, transitions and others.
  • Mark: select state/transition for set operation and partial model execution. Also supports left-mouse lasso to select multiple states.

Double click on canvas backgroud reset to Edit mode.

Four easy steps to create model:
  • Create Initial State - select O, click on canvas to place Initial State
  • Create Final State - select , click on canvas to place Final State
  • Create AUT states - select , click on canvas to place state on the model
  • Create Transitions - select ➜, click on source state and target state
Other common actions while in Edit mode:
  • Move states - hover over state, click on 👆 to activate move and click again to place the state on desired location
  • Adjust transitions - click on transition segment to activate mode and click again at desired location
  • Relocate transition ends - move transition end segment to edge of the state and click again
  • Move transition label - hover over transition label, click on 👆 to activate move and click again to place label on desired location
  • State/Transition menus: hover over state / transition label, click to show circular menu items
  • State/Trans Properties: double click on state/transtion label or click on state/transition menu item
