Write groovy scripts to output test cases or perform test automation as model executes.

There are 4 types of scripts you may write:

  • TRIGGERS: script for each state and transition to be executed when it's traversed. Ctrl/Cmd-I to add a script.
  • PAGES: UI page object scripts (optional) to be used in TRIGGERS script.
  • STEPS: Behavior Driven Development (BBD) style step definition to be used in TRIGGERS script.
  • Custom Script: custom script - utility functions to be called in TRIGGERS script.

Script editor supports Code-Assist on predefined script variables ($SYS, $UTIL, $VAR and $RAND). Ctrl/Cmd-Space to bring up Code Assist.

Requirements: click on Req to view and edit requirements to be covered by this model.
