Partnerships and Integrations

TestOptimal has established partnership and/or developed product integration with the following QA solutions and tools.
XQual delivers XStudio, a test management application that handles the complete life-cycle of QA/testing projects from end to end: users, requirements, specifications, scrum project management, SUTs, tests, test plans, test reports, test campaigns and bugs. XStudio allows scheduling or running fully automated tests (including support for TestOptimal) or manual test campaigns.
Selenium is a suite of tools to automate web app testing across many platforms. It is one of the most popular open source test framework for web applications.
HtmlUnit HtmlUnit is a "GUI-Less browser for Java programs". It models HTML documents and provides an API that allows you to invoke pages, fill out forms, click links, etc... just like you do in your "normal" browser.
White is a framework for automating rich client applications based on Win32, WinForms, WPF, Silverlight and SWT (Java) platforms. It is .NET based and does not require the use of any proprietary scripting languages.
AutoIt v3 is a freeware BASIC-like scripting language designed for automating the Windows GUI and general scripting. It uses a combination of simulated keystrokes, mouse movement and window/control manipulation in order to automate tasks in a way not possible or reliable with other languages (e.g. VBScript and SendKeys). AutoIt is also very small, self-contained and will run on all versions of Windows out-of-the-box with no annoying "runtimes" required!
